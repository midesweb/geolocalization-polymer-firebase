# Firebase Demo with Polymer + HTML5 Geolocation

This is an app based on Polymer Starter Kit, developed to show how to work with Firebase service. It uses the Polymer google-map custom elements to put a mark inside a Google Map in all users location.

As each user entered in the app, new mark shown in the map. As real time application, you do not need to refresh your browser to see new user marks. The Firebase data source do all the work for you.
